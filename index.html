Initial Connection of a Github Repo with the Local Environment

  1. git init
  2. git add .
  3. git commit -m "first commit"
  4. git branch -M main
  5. git remote add origin (repo address)
  6. git push -u origin main

How to push code to an existing github repo

  1. git add .
  2. git commit -m "meaningful message"
  3. git push
